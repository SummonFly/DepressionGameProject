# Abyss of Despair

**Abyss of Despair** — это 2D-нарративная игра, созданная в Unity, которая исследует тему психического здоровья через метафоры, головоломки и атмосферное повествование. Игрок помогает главному герою пройти через сложный эмоциональный период, решая пазлы и взаимодействуя с окружением, шаг за шагом находя путь к внутреннему свету.

> *Это командный учебный проект, выполненный в рамках образовательной программы.*


Внимание
> *Этота игра находиться в разработке и доступна только сырая версия*

---

## Ключевые особенности игры

*   **Проработанная тематика:** Сюжет, построенный на метафорическом преодолении депрессии.
*   **Атмосферный мир:** Мрачная, но детализированная пиксельная графика, создающая уникальное настроение.
*   **Разнообразные механики:**
    *   Классическое платформерное управление и исследование локаций.
    *   Интерактивные диалоги для раскрытия сюжета.
    *   **8 уникальных пазлов** и **6 головоломок "Пятнашки"**, интегрированных в геймплей.
    *   3 сюжетные кат-сцены (видео).
*   **Полноценный цикл:** 6 локаций, 14 интерактивных объектов, система меню и сохранений.

---

## Технологический стек (как разрабатывалось)

*   **Движок:** Unity
*   **Язык программирования:** C#
*   **Графика:** Пиксель-арт
*   **Управление:** Клавиатура (WASD для движения, E для взаимодействия) и мышь.
*   **Версионный контроль:** Git, GitHub
*   **Командная организация:** Распределение ролей (геймдизайн, программирование, графика, нарратив).

## Структура проекта (для разработчиков)

Основные директории проекта Unity:

*   Assets/
    *   Audio/                 # Звуковые эффекты и музыкальное сопровождение
    *   InputSystem/           # Конфигурации системы ввода
    *   UnityObjects/               # Игровые объекты, префабы, модели
    *   Plugins/               # Сторонние плагины и библиотеки
    *   PostProcessing/        # Настройки пост-обработки изображения
    *   Resources/             # Ресурсы, загружаемые во время выполнения
    *   Scenes/                # Игровые сцены
    *   TextMesh Pro/          # Ресурсы для продвинутой текстовой системы
    *   Textures/              # Текстуры и спрайты
    *   Timeline/              # Анимации и кат-сцены на основе Timeline


**Примечание:** Основная игровая логика на C# располагается в соответствующих папках рядом с префабами и сценами.

## Как запустить игру

### Для игроков (из готовой сборки)

1.  Перейдите в раздел Releases этого репозитория (или в папку `Builds/`).
2.  Скачайте архив для вашей операционной системы (Windows).
3.  Распакуйте архив и запустите файл `DepressionGame.exe`.

### Для разработчиков (из исходного кода)

1.  Убедитесь, что у вас установлен Unity Hub и редактор Unity.
2.  Склонируйте репозиторий:
    ```bash
    git clone https://github.com/SummonFly/DepressionGameProject.git
    ```
3.  Откройте проект в Unity через Unity Hub.
4.  Откройте главную сцену и нажмите "Play".
